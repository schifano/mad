The Idea

  • Ergosophere is a study application on iOS
  • Integrates Jawbone UP 24 live syncing technology to track idle time - push notifications to study
  • Uses learning strategies to reinforce info
  • Creates fun study sessions that are space-themed, flat UI design
  • A very tiny piece of what actually works [Demo]
    • There is so much more behind the scenes
  • API [Demo]

These are not the [requirements] you are looking for.

  • iOS core data? No.
    • Storing data locally or in two places will not cut it
    • Online server / database as single source of storage & syncing
    • L(inux) A(pache) M(ySql) P(HP)
    • API is important!

If we build, WE will learn.

  1. Big infrastrucutre - not just an iOS app
  2. Server - phones don't magically receive data
  3. APIs should be viewed as important as 50% of the entire solution
  4. Objective-C, xCode and iOS
    • Design the interface
    • Tightly integrate code
    • Create universal iPhone & iPad app
  5. We are humans. Student humans.

What we did right

  • Found resources
    • Code School - Try iOS & Objective-C
    • App Conda tutorials
  • Leveraged our strenghts - two sides of the same coin
  • Researched ahead of time - we're in this for the long haul
  • Had patience - you need lots of it - be patient and celebrate the small wins.
  • Be courageous

Final Words

We'll be back. Give us a year, we're not giving up.
